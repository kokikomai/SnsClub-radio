議事録AI自動生成ツール

山田さん

2週間前に公開

デモ

デモ動画を再生

概要

会議の音声を録音するだけで、AIが自動的に議事録を生成するツールです。要約、アクションアイテムの抽出、参加者の発言分析まで行います。

主な機能

  • 音声の自動文字起こし（Whisper API使用）
  • AIによる要約生成（GPT-4使用）
  • アクションアイテムの自動抽出
  • 話者ごとの発言の分離
  • Slack/Teams連携
  • Notion/Google Docsへの自動エクスポート

技術スタック

  • フロントエンド: Next.js, Tailwind CSS
  • バックエンド: Python (FastAPI)
  • AI: OpenAI Whisper, GPT-4
  • インフラ: Vercel, Railway

使い方

  1. 会議の開始時にツールを起動
  2. 「録音開始」ボタンをクリック
  3. 会議終了後「録音停止」をクリック
  4. 数分後に議事録が自動生成されます
